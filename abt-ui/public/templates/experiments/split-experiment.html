Split URL Experiment

An experiment is a new change you want to test. Please provide a Name and URL of the page you want to run the experiment on.

Variations

A Variation is a different design of your page. Please provide the URL of that new design and assign it some percentage of traffic.
  • The Control variation is your original experiment page.
  • You have to host these new designs on your server before creating this experiment.
Example : If your experiment page URL is http://yourdomain.com/index.html then your variations could be :
Variation #1 : http://yourdomain.com/index_v2.html
Variation #1 : http://yourdomain.com/index_v3.html
